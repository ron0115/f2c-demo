# How to run this Project?

## Install

```sh
# or pnpm or yarn
pnpm i
```

## Install plugin

```
pnpm i f2c-server -D
```

## Run React Project

```sh
npm run dev
```

## Run F2C server

```sh
# the command use `outFolder=GeneratedComponent` to override figma plugin's path setting
npm run f2c
```
